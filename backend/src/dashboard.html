    🩺 Vibe Clinic 치료 가이드
    
    
    
      
    
        
    1
    
        
    
          
    프로젝트 선택 및 이동
    
          
    좌측 패널에서 대상 프로젝트 폴더를 선택하고 이동을 누릅니다. 만약 진단 폴더(.vibe-clinic)가 없는 경우 진단 도구 설치 (초기화) 버튼을 눌러 검진 기본 환경을 구축하세요.
    
        
    
      
    
      
    
        
    2
    
        
    
          
    진단 노드 구조화
    
          
    프로젝트가 스캔되면 우측 캔버스에 [SYSTEM ──> TASK ──> FUNCTION] 계층으로 진단 노드들이 동적 마인드맵 형태로 시각화됩니다.
    
        
    
      
    
      
    
        
    3
    
        
    
          
    진단 실행 및 분석
    
          
    우측 상단의 '진단 실행' 버튼을 누릅니다. 통과하지 못한 노드는 흐릿하거나 느낌표 표시가 뜨며, 해당 노드를 클릭하면 우측 슬라이딩 드로어를 통해 상세 에러 및 패치 코드를 검토할 수 있습니다.
    
        
    
      
    
      
    
        
    4
    
        
    
          
    AI 수리 및 실시간 히스토리 확인
    
          
    수리 제안을 승인하여 완치(OK)되면 해당 노드 카드에 치료 이력 타임라인이 누적 기록되어 코드가 실제 어떻게 수정되었는지 투명하게 파악 가능합니다.
